## ğŸ”‘ Key Components

### Document Processing
- Automatic text extraction from PDFs
- Intelligent document type detection (fees worksheet,contract,payslip)
- Smart chunking with page number preservation
- Metadata enrichment for better retrieval

### RAG Pipeline
- **Retrieval**: FAISS-powered semantic search finds relevant document chunks
- **Augmentation**: Context from top-K similar chunks
- **Generation**: Gemini LLM synthesizes natural language answers
- **Grounding**: Responses include source page numbers

### Prompt Engineering
Custom prompts ensure:
- Answers are grounded in document content
- Explicit mention of source documents and pages
- Acknowledgment when information isn't available : No hallucinations.
- Concise yet complete responses
